Desktop App

toT

toT is a desktop app that turns text into estimated reading time and helps you plan realistic reading workloads.

It combines file text import/extraction from text and image files, configurable WPM presets, precise counting, text snapshots, a task editor, and a stopwatch so you can measure, organize, and complete readings with less guesswork.

Google Drive access is used for OCR when you extract text from scanned PDFs or images.
