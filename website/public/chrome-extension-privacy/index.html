Política de privacidad — Extensión Chrome de toT

Fecha de vigencia: 2026-04-24

Esta política se aplica únicamente a la extensión Chrome toT — Tiempo de lectura. No describe la app de escritorio toT ni sus funciones de importación, OCR, snapshots, tareas u otras herramientas.

1. Finalidad de la extensión

La extensión tiene una única finalidad: estimar el tiempo de lectura del texto seleccionado por el usuario en páginas web.

Para hacerlo, la extensión puede leer el texto que el usuario selecciona en páginas http y https, contar sus palabras localmente y mostrar una estimación de tiempo de lectura en un pequeño panel dentro de la página.

2. Datos que procesa

La extensión procesa localmente:

  • el texto seleccionado por el usuario en una página web;
  • la preferencia de WPM, es decir, palabras por minuto;
  • el estado activado o desactivado por sitio.

El texto seleccionado se usa solo para calcular el número de palabras y el tiempo estimado de lectura. La extensión no guarda un historial de selecciones.

3. Almacenamiento local

La extensión usa chrome.storage.local para guardar:

  • la preferencia de WPM del usuario;
  • la lista local de sitios donde la extensión fue desactivada por el usuario.

Estos datos permanecen en el navegador del usuario. No se envían al desarrollador ni a servidores externos.

4. Conectividad y transferencia de datos

La extensión no transmite el texto seleccionado, el contenido de las páginas, el historial de navegación ni las preferencias guardadas a servidores externos.

La extensión no usa analítica, telemetría, publicidad, tracking, profiling ni SDKs de terceros.

La extensión no vende ni transfiere datos de usuario.

La extensión no usa datos de usuario para fines no relacionados con la estimación del tiempo de lectura del texto seleccionado.

La extensión no usa datos de usuario para evaluar solvencia, conceder préstamos, publicidad, tracking ni profiling.

5. Permisos

La extensión solicita el permiso storage para guardar preferencias locales.

También ejecuta content scripts en páginas http y https para poder detectar texto seleccionado por el usuario y mostrar el panel de estimación de tiempo de lectura. Ese acceso se usa solo para la finalidad descrita en esta política.

6. Código remoto

La extensión no usa código remoto. El código JavaScript necesario para su funcionamiento está incluido en el paquete de la extensión.

7. Control del usuario

El usuario puede:

  • cambiar el valor de WPM desde el panel de la extensión;
  • activar o desactivar la extensión por sitio desde el popup;
  • eliminar los datos locales de la extensión desde la configuración de Chrome o eliminando la extensión.

8. Cambios

Si la extensión incorpora nuevas formas de tratamiento de datos, esta política será actualizada antes o junto con la publicación de esa nueva versión.

9. Contacto

Para dudas, reportes o solicitudes relacionadas con esta política, escribe a cibersino@gmail.com o usa el repositorio oficial del proyecto.

Contacto: GitHub Issues o cibersino@gmail.com.
