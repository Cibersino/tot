Privacidad y datos

  • Telemetría: no se recopilan métricas ni se envían datos de uso.
  • Texto del usuario: el contenido que ingresas/pegas en la app no se envía a Internet ni se comparte con servidores de los desarrolladores (se procesa localmente).
  • Almacenamiento: la configuración y el estado se guardan localmente en el equipo del usuario.
  • Actualizaciones: la app solo consulta información pública de versiones y abre el navegador para descargar; no descarga ni instala actualizaciones automáticamente.
  • Enlaces externos: la app también puede abrir enlaces externos por acción explícita del usuario (por ejemplo, desde páginas informativas o desde el Editor de Tareas).
  • OCR: si el usuario decide usar la función OCR para extraer texto de archivos, su activación abre el navegador del sistema para autorizar Google y los archivos elegidos para OCR se envían a Google solo para esa operación; no se envían a los desarrolladores. El usuario puede revocar la autorización mediante Menú > Preferencias > Desconectar Google OCR.

Conectividad: la app sigue siendo principalmente local para escribir/pegar/editar texto y calcular conteos/estimaciones. La conectividad normal incluye verificación de actualizaciones y apertura de enlaces externos por acción del usuario. Si el usuario decide activar o usar la función OCR, también existe conectividad con Google para autorización y procesamiento OCR de archivos seleccionados por el usuario.

Para más detalles, ver el documento de PRIVACY.md.
